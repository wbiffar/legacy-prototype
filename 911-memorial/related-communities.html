Concept A: Stacked Carousels

Full-width section between the hero and main content. Each carousel is labeled by relationship type (topic, geography, trending). Shows 3–4 cards per row on desktop, 1.5 visible on mobile with peek-to-scroll.

✓ High visibility ✓ Multiple lists visible at once ✓ Proven pattern (Netflix, Facebook) ✗ Pushes main content down ✗ Could feel heavy

Concept B: Sidebar Widgets

Compact card widgets stacked in the existing right sidebar (below Charity and Memorial Tree callouts). Each widget shows 3–5 community links with minimal metadata. Leverages existing layout infrastructure with zero disruption to main content.

✓ Minimal & non-distracting ✓ Leverages existing sidebar ✓ Always visible on desktop ✗ Hidden on mobile ✗ Limited 300px width

Concept C: Inline Teaser + Expandable Drawer

A slim inline teaser banner with minimal footprint on the page. Clicking "Explore" opens a slide-up drawer with tabbed navigation (By Topic, By Location, Popular, For You). Respects the memorial context by making discovery entirely opt-in.

✓ Minimal page footprint ✓ Opt-in discovery ✓ Tabs organize relationship types ✓ Great on mobile (full-screen) ✗ Lower discoverability ✗ Extra click to see content

★ Recommended: Hybrid (B + C)

Sidebar widgets on desktop for always-visible, minimal discovery. On mobile (where the sidebar is hidden), an inline teaser opens a full-screen drawer with tabbed navigation. Best of both worlds: respects the memorial context while ensuring mobile users can still discover communities.

✓ Minimal on desktop (sidebar) ✓ Full discovery on mobile (drawer) ✓ Multiple lists visible ✓ Extensible for signed-in "For You" ✓ No layout changes needed

Concept D: Featured Groups (Existing Component)

Reuses the existing "Featured Groups" component from the Search/Listing pages, dark-adapted for the memorial theme. Zero net design debt — users recognize the pattern across the site. The bookmark icon provides a native save affordance, and the badge system (Featured, New, Trending) handles editorial curation naturally. Toggle between Sidebar and Inline placements above.

✓ Reuses approved existing component ✓ Consistent UX across listing + memorial pages ✓ Bookmark affordance built in ✓ Badge system handles editorial curation ✗ Sidebar invisible on mobile ✗ Lower discovery pressure than carousels
